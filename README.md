# SciPy 2026 tutorial on [Reproducible CUDA Accelerated Workflows for Scientists with Pixi](https://matthewfeickert-talks.github.io/reproducible-cuda-workflows-with-pixi-scipy-2026/)

## Development

1. Install [Pixi](https://pixi.prefix.dev/)
2. Clone this repository and navigate to the top level
3. Run

```
pixi run start
```

4. Edit files under `book/`

For reference on MyST functionality and features reference the [MyST documentation](https://mystmd.org/guide/).
